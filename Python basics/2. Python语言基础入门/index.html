    1.3 字面量

    在代码中被写下来的固定的值.

    • 列表、元组、集合、字典（此处后续会讲到）

    1.4 注释

    单行注释

    # 一个#号表示单行注释，与后面的文本注释以一个空格隔开
print("hello world")

    多行注释

    """
    用三个双引号
    表示
    多行注释
"""
print("hello")

    1.6 变量

    储存计算结果或能表示值的抽象概念.

    变量名称 = 变量的值

    # python 变量相关操作

# 定义一个变量，记录钱包余额
money = 50
# 通过print语句，输出变量记录的内容
print("钱包还有：", money)

# 花费10元
money = money - 10
print("花费10元，还剩余：", money, "元")

    复制代码可以使用快捷键CTRL +D.

    1.7 数据类型

    第一阶段只接触字符串，整型，浮点型.

    # 判断数据类型以及type()语句的使用

print(type("wayne"))
print(type(123))
print(type(13.14))

# type查看变量中储存的数据类型

name = "hello"
name_type = type(name) # 查看数据的类型
print(name_type)
    输出如下结果： 
    <class 'str'>
<class 'int'>
<class 'float'>
<class 'str'>

    1.8 数据类型转换

    常见的转换语句： 

    int(x) #将x转换为一个整数
float(x) #将x转换为一个浮点数
str(x) #将对象x转换为字符串

    （与之前的type()语句相同，可以用print直接输出）

    # 数据类型转换
a = str(11)
print(type(a), a)

b = str(13.14)
print(type(b), b)

# 将字符串转换成数字
c = int("666")
print(type(c), c)

d = float("66.6")
print(type(d), d)

# 浮点数转整数
p = int(11.545)
print(type(p), p)


    <class 'str'> 11
<class 'str'> 13.14
<class 'int'> 666
<class 'float'> 66.6
<class 'int'> 11
    浮点数转整数会丢失精度.

    1.11 字符串扩展

    # 单引号字符串
name = 'wayne'
print(type(name), name)
# 双引号字符串
name = "wayne"
print(type(name), name)
# 三引号字符串
name = """
bruce
wayne
"""
print(type(name), name)

# 转义字符的使用
name = "\"hello\""
print(type(name), name)

    <class 'str'> wayne
<class 'str'> wayne
<class 'str'> 
bruce
wayne

<class 'str'> "hello"


    # 字符串拼接
name = "wayne"
address = "西北工业大学"
tel = "15907218668" #整型需要加双引号才能与字符串相接
print("我是" + name + "来自" + address + tel)

    我是wayne来自西北工业大学15907218668


    # 占位的方式完成拼接
name = "西北工业大学"
message = "世界顶尖院校：%s" % name
print(message)

# 完成数字与字符串的拼接
class_num = 2022
avg_salary = 2022301578
message = "西北工业大学机电学院%s级本科生，学号%s" % (class_num, avg_salary)
print(message)

    世界顶尖院校：西北工业大学
西北工业大学机电学院2022级本科生，学号2022301578

    • %s 将内容转换为字符串

    • %d 将内容转换成整数

    • %f 将内容转换成浮点型

    # 格式精确化
num1 = 11
num2 = 11.345
print("%5d" % num1)
print("%1d" % num1)
print("%7.2f" % num2)
print("%.2f" % num2)

       11
11
  11.35
11.35


    # 第二种字符串的格式化方法：f{占位}

name = "wayne"
year = 2004
# f: format
print(f"我是{name},{year}出生")

# 对表达式进行字符串格式化

print("1 * 1的结果是：%d" % (1 * 1))
print(f"1 * 2的结果是：{1 * 2}")
print("字符串在python中的类型名是：%s" % type("字符串"))

    我是wayne,2004出生
1 * 1的结果是：1
1 * 2的结果是：2
字符串在python中的类型名是：<class 'str'>

    1.12 数据输入

    使用input语句实现数据输入. 

    # 使用input语句获取键盘的输入信息
print("give me a number")
name = input()
print("your number is %s" % name)

# 程序的改进：input(提示信息)，可以用在使用者输入内容之前显示提示信息.
name = input("give me a number")
print("your number is %s" % name)

# input语句中的数据类型
message = input("give me a message")
print("your message type is %s", type(message))

# 数据类型的转换
num = input("give me a number")
num = int(num)
print("the type is: ",type(num))

    give me a number
3
your number is 3
give me a number
3
your number is 3
give me a message
123
your message type is %s <class 'str'>
give me a number
123
the type is:  <class 'int'>
    注意：input语句默认接收的类型都是字符串！

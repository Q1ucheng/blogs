    1.3 字面量

    在代码中被写下来的固定的值.

    字面量的分类

    • 数字（整数，浮点数，复数，布尔）
    • 字符串（由任意数量的字符组成，字符串需要用双引号包括起来）
    • 列表、元组、集合、字典（此处后续会讲到）
    print(123)
print(13.14)
print("Wayne's blogs")

    1.4 注释

    单行注释

    # 一个#号表示单行注释，与后面的文本注释以一个空格隔开
print("hello world")

    多行注释

    """
    用三个双引号
    表示
    多行注释
"""
print("hello")

    1.6 变量

    储存计算结果或能表示值的抽象概念.

    变量名称 = 变量的值

    # python 变量相关操作

# 定义一个变量，记录钱包余额
money = 50
# 通过print语句，输出变量记录的内容
print("钱包还有：", money)

# 花费10元
money = money - 10
print("花费10元，还剩余：", money, "元")

    复制代码可以使用快捷键CTRL +D.

    1.7 数据类型

    第一阶段只接触字符串，整型，浮点型.

    # 判断数据类型以及type()语句的使用

print(type("wayne"))
print(type(123))
print(type(13.14))

# type查看变量中储存的数据类型

name = "hello"
name_type = type(name) # 查看数据的类型
print(name_type)

    输出如下结果：

    <class 'str'>
<class 'int'>
<class 'float'>
<class 'str'>

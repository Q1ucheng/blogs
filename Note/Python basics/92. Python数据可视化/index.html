    json数据格式

    json是一种在各个编程语言中通用的数据格式，负责不同编程语言中的数据传递和交互

    一个简单的演示： 

    # JSON数据和Python字典的相互转换
import json
# 准备列表，转换为json
data = [{"name": "wayne", "age": 19}, {"name": "bruce", "age": 18}, {"name": "jack", "age": 22}]
json_str = json.dumps(data)
print(type(json_str))
print(json_str)

    pyecharts模块

    pyecharts官方文档

    简单应用pyecharts

    # pyecharts的使用
# 导包
from pyecharts.charts import Line
from pyecharts.options import TitleOpts, LegendOpts, ToolboxOpts, VisualMapOpts
# 创建一个折线图对象
line = Line()
# 添加折线图x轴数据
line.add_xaxis(["A", "B", "C"])
# 添加折线图y轴数据
line.add_yaxis("GDP", [30, 20, 10])

# 设置全局配置项set_global_opts来设置
line.set_global_opts(
    title_opts=TitleOpts(title="GDP展示", pos_left="center", pos_bottom="1%"),  # 设置标题,居中,靠近底部
    legend_opts=LegendOpts(is_show=True),  # 显示图例
    toolbox_opts=ToolboxOpts(is_show=True),  # 显示工具箱
    visualmap_opts=VisualMapOpts(is_show=True),  # 控制视觉映射
)
# 将代码生成为图像
line.render()

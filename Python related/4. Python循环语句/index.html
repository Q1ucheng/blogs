    # 打印九九乘法表

    j = 1
    while j <= i:
        p = i * j
        print(f"{i} * {j} = {p}\t", end='')
        j += 1
    i += 1
    print() #print空内容以实现换行

    3. for 循环

    3.1 for循环基础语法

    for 临时变量 in 待处理数据集(序列类型)：
    循环满足条件时执行的代码

    数数有几个？

    定义字符串变量name， 统计有多少个英文字母“a”，内容为："my name is Wayne"

    # for循环练习
name = "my name is Wayne"
cnt = 0
for x in name:
    if x == "a":
        cnt += 1
print(f"my name is Wayne中有{cnt}个a")

    my name is Wayne中有4个a

    4. for循环的嵌套

    Note

    用for循环嵌套打印九九乘法表.

    # 用for循环打印九九乘法表
i = 1
j = 1
for i in range(1, 10):
    for j in range(1, i+1):
        p = i * j
        print(f"{i} * {j} = {p}\t", end='')
    print()

    6. 循环综合练习

    发工资

    某公司账户有10000元，给20名员工发工资。

    1. 员工编号1到20，从编号1开始，一次领取工资，每人可领取1000元

    2.领工资时，判断员工的绩效分（1~10随机生成），低于5不发工资，换下一位。

    3.工资发完就结束。

    4.打印出发工资的情况。 

    # 发工资
import random
p = 10000
for x in range(1, 21):
    num = random.randint(1, 10)
    if num < 5:
        print(f"员工{x},绩效分{num},低于5,不发工资")
        continue
    if num >= 5:
        p = p - 1000
        print(f"向员工{x}发放工资1000元,账户余额{p}")
        if p == 0:
            print("工资发完了")
            break
